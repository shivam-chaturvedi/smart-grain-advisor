# Smart Grain Advisor — Frontend (Vite + React)

## Setup

```bash
cd /Users/shared/shivam/naysa/smart-grain-advisor-fe
npm install
```

## Configure backend URL

Create `.env` (or copy from `.env.example`):

```bash
cp .env.example .env
```

Default:
- `VITE_API_BASE_URL=http://localhost:5050`

## Run

```bash
npm run dev
```
